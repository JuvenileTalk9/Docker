# Docker勉強資料

## 前提
- [VirtualBox](https://www.virtualbox.org/)上のCentOS7.6で勉強しています
- Dockerバージョンは19.03.8（build afacb8b）です
- Docker初心者がDockerでWebアプリケーションのCIができるようになることが目標です
- 余裕があれば[Kubernetes](https://kubernetes.io/)にも挑戦します
